# Plagiarism Checker (Python)


This project is a **basic and simple plagiarism checker** written in Python. It scans text files in the directory and computes similarity between them to detect potential plagiarism. The current version uses dummy files for demonstration and basic vector-based similarity.


## Features

- Compare text files for similarity
- Simple, easy-to-understand Python code
- Uses [scikit-learn](https://scikit-learn.org/) for vectorization and similarity calculation
- Dummy files included for testing



## Getting Started

### Prerequisites

- Python 3.x
- `scikit-learn` library

### Installation

Install the required Python package:

```bash
pip install scikit-learn
```

### Usage

1. Place your text files in the project directory.
2. Run the Python script to check for plagiarism between files.
3. Review the similarity scores to identify potential matches.



## Project Structure

- `plagiarism_checker.py` — Main script for checking plagiarism
- `dummy_files/` — Example text files for testing
