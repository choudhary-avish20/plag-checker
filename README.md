# Plagiarism Checker (Python) Version -2


This project is a **basic and simple plagiarism checker** written in Python. It scans text files in the directory and computes similarity between them to detect potential plagiarism. The current version uses dummy files for demonstration and basic vector-based similarity.


## Features for version 2

- Much larger source files
- embedding creation with sentence transformers
- planned implementation of FAISS, SQL storage,Top-K retrieval
- Simple and handy Front-end for testing.



## Source Data

The corpus is built from compiled arXiv source data along with supporting text from Kaggle and a Wikipedia dump. It includes:

- arXiv source archives (https://www.kaggle.com/datasets/Cornell-University/arxiv)
- Wikipedia dump text for broader cover (https://dumps.wikimedia.org/enwiki/20260601/)

This combined dataset is used for plagiarism testing and similarity comparison in the project.
